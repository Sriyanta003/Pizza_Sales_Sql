# Pizza_Sales_Sql

This project analyzes pizza sales data using SQL to identify sales trends, top-selling pizzas, and overall revenue performance. The analysis helps generate useful business insights from the sales dataset.

🍕 Pizza Sales Analysis using SQL

📊 Overview

This project analyzes pizza sales data using SQL to gain insights into business performance. The goal is to understand customer ordering patterns, identify popular pizzas, and evaluate overall sales and revenue.

📁 Dataset

The dataset contains information about:

- Orders
- Order details
- Pizza types
- Pizza prices
- Pizza categories and sizes

These tables are connected using primary and foreign keys to perform detailed analysis.

🛠 Tools & Technologies

- SQL (MySQL / PostgreSQL / SQL Server)
- SQL Joins
- Aggregations (SUM, COUNT)
- GROUP BY & ORDER BY
- Subqueries

🔍 Analysis Performed

- Total number of orders
- Total revenue generated
- Highest priced pizza
- Most common pizza size ordered
- Top 5 most ordered pizzas
- Sales distribution by pizza category

📈 Key Insights

- Identified the most popular pizza types among customers.
- Analyzed revenue contribution from different pizza categories.
- Found the most frequently ordered pizza size.
- Discovered top-selling pizzas driving business revenue.

🚀 Conclusion

This project demonstrates how SQL can be used to transform raw sales data into meaningful insights that help businesses make data-driven decisions.
